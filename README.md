# 투두리스트

## 🖥 프로젝트 소개
- 할일 목록(todo-list) 웹 어플리케이션입니다.

## ⚙ 개발환경
- 개발 도구: IntelliJ IDEA
- 언어: Java
- 프레임워크: Spring Boot
- DB: MySQL
- 빌드 도구: Gradle

## 📌 주요 기능
- 로그인
- 회원가입
- 투두리스트 - 아이템 보기, 등록, 수정, 삭제
